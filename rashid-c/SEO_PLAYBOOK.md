# SEO Playbook

This checklist helps improve ranking for `Rashid`, `Rashidc`, `Rashid C`, `Rashid_C`, `Rashid-C`, and job-intent searches in GCC, Dubai, Saudi Arabia, Kerala, and India.

## 1) Search Console Setup (Google)
1. Go to Google Search Console.
2. Add property: `https://rashidc.site/`.
3. Verify ownership:
- Recommended: DNS TXT record.
- Alternative: HTML tag in [index.html](/d:/Program files/Mern_Projects/Portfolio/rashid-c/index.html) head.
4. Submit sitemap: `https://rashidc.site/sitemap.xml`.
5. Request indexing for homepage after each major update.

## 2) Bing Webmaster Setup
1. Add site in Bing Webmaster Tools.
2. Verify ownership (DNS or meta tag).
3. Submit sitemap: `https://rashidc.site/sitemap.xml`.
4. Use URL Inspection to request crawl for key pages.

## 3) Verification Placeholders
When needed, add these tags into [index.html](/d:/Program files/Mern_Projects/Portfolio/rashid-c/index.html) inside `<head>`:

```html
<meta name="google-site-verification" content="PASTE_GOOGLE_CODE" />
<meta name="msvalidate.01" content="PASTE_BING_CODE" />
```

## 4) Indexing Workflow (After Deploy)
1. Deploy site.
2. Open `/robots.txt` and `/sitemap.xml` in browser.
3. Validate canonical URL in page source.
4. Submit sitemap in Google + Bing.
5. Request indexing of homepage and major sections.

## 5) Monthly SEO Tasks
1. Add at least 1 new project or major update.
2. Refresh project descriptions with searchable keywords.
3. Add/update backlinks:
- LinkedIn profile
- GitHub bio and pinned repos
- Resume/profile platforms
4. Check GSC coverage and fix crawl/index errors.
5. Review top search queries and update copy accordingly.

## 6) On-Page Content Targets
Use these terms naturally (not keyword stuffing):
- Full-Stack Software Developer
- MERN Stack Developer
- Next.js Developer
- Software Developer in Dubai / GCC / Saudi Arabia / Kerala / India
